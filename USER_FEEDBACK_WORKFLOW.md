# User Feedback Workflow

## Bug Reports

Use: `BUG_REPORT_TEMPLATE.md`

Submit via: GitHub Issues (template) or email to feedback@dawaisaver.pk

## Feature Requests

Use: `FEATURE_REQUEST_TEMPLATE.md`

Submit via: GitHub Issues (template) or email to feedback@dawaisaver.pk

## Feedback Triage

1. All feedback reviewed weekly
2. Bugs prioritized by severity
3. Features evaluated for roadmap alignment
4. Contributors acknowledged in CHANGELOG.md
